Acerca de Renzo Anaya

Soy Renzo Anaya, un Fullstack Developer apasionado por la tecnología y el diseño. A lo largo de mi carrera, he adquirido habilidades en:

  • HTML
  • CSS
  • Tailwind CSS
  • JavaScript
  • TypeScript
  • React
  • Next.js
  • Python
  • Node.js
  • SQL
  • MongoDB

Además de mis habilidades de desarrollo, tengo experiencia en diseño, específicamente utilizando Figma para diseñar aplicaciones. El diseño es esencial para crear experiencias de usuario atractivas y funcionales, y siempre estoy buscando formas de mejorar y innovar en este campo.

He tenido el privilegio de liderar varios proyectos para diferentes empresas y he creado sitios web y aplicaciones para mis propias empresas, como Repo, Wiqli y Alchemia Caffè. Cada proyecto ha sido una oportunidad para aprender, crecer y aportar valor a mis clientes y usuarios.
